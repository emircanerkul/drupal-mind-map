* [UIkit Components](https://www.drupal.org/project/uikit%5Fcomponents): Provides additional components and functionality to the UIkit base theme that cannot be provided through the frontend, such as Views plugins.
* [UIkit Image formatter](https://www.drupal.org/project/uikit%5Fimage%5Fformatter): Provide 3 new formatters for Image Field, based on UIkit 3 library : Lightbox (gallery), Slideshow and Slider (carousel)
* [UIkit Iconpicker ](https://www.drupal.org/project/uikit%5Ficonpicker): this module is an utility to manage an UIkit 3 icon Field.
