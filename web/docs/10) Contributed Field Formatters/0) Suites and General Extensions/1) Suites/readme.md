These modules provide field formatter plugins for a large number of core field types. The field formatters extend core field formatters in a similar way.

* [Barcodes](https://www.drupal.org/project/barcodes)
* [bxSlider](https://www.drupal.org/project/bxslider)
* [Element Class Formatter](https://www.drupal.org/project/element%5Fclass%5Fformatter)
* [Entity Class Formatter](https://www.drupal.org/project/entity%5Fclass%5Fformatter)
* [Fallback Formatter](https://www.drupal.org/project/fallback%5Fformatter)
* [Field Limiter](https://www.drupal.org/project/field%5Flimiter)
* [Field Slideshow](https://www.drupal.org/project/field%5Fslideshow)
* [Formatter Suite](https://www.drupal.org/project/formatter%5Fsuite)
* [Views Field Formatter](https://www.drupal.org/project/views%5Ffield%5Fformatter)
