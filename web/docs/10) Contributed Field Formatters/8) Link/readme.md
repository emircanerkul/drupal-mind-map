Drupal 8 ships with two field formatters for link fields: "Link" and "Separate Link Text and Url". The following contributed modules provide additional field formatters plugins for link fields.

* [Button Link Formatter](https://www.drupal.org/project/button%5Flink)
* [Fixed Text Link Formatter](https://www.drupal.org/project/fixed%5Ftext%5Flink%5Fformatter)
* [Formatter Suite](http://www.drupal.org/project/formatter%5Fsuite)
* [Link Plain Text Formatter](https://www.drupal.org/project/link%5Fplain%5Ftext%5Fformatter)
* [Swagger UI Field Formatter](https://www.drupal.org/project/swagger%5Fui%5Fformatter)
