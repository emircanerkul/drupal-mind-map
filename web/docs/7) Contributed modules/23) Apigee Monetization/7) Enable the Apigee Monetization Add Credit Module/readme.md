Enable prepaid app developers to add credit to their balance by installing the Apigee Monetization Add Credit module, setting up a Drupal Commerce store, and defining products to increment the credit balance. The steps are outlined in the following sections:

* [Step 1: Install the required Monetization and Commerce modules](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-1-install-the-required-monetization-and-commerce-modules)
* [Step 2: Set up a store](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-2-set-up-a-store)
* [Step 3: Add products to the store](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-3-add-products-to-the-store)
* [Step 4: Add an Add Credit button to your prepaid balance page](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-4-add-an-add-credit-button)
* [Step 5: Configure permissions to view and purchase Add Credit products](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-5-configure-permissions-for-developers)
* [Step 6: Configure wait settings for Add Credit top-ups](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-6-configure-wait-settings)
* [Step 7: Configure notifications for adding credit](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-7-configure-notifications-for-adding-credit)
* [Step 8: Test the shopping experience](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-8-test-the-shopping-experience)
* [Step 9: Manage orders](https://www.drupal.org/docs/8/modules/apigee-monetization/support-adding-credit-to-a-prepaid-balance#step-9-manage-orders)
