The Allow a content type only once (Only One) for Admin Toolbar module handles modifications to the [Admin Toolbar Extra Tools](https://www.drupal.org/project/admin%5Ftoolbar) menu entries related to the configured content types.

There is no configuration required. When a content type is configured to have Only One node and a node exists for this content type, the content type name will be shown with the text "(Edit)" at the end in the menu:

_Content » Add content_
