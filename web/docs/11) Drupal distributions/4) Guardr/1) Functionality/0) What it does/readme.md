Guardr pre-packages the following modules, activates most of them, and sets pre-configured, secure values for some.

* [autologout](https://drupal.org/project/autologout)
* [clear\_password\_field](https://drupal.org/project/clear%5Fpassword%5Ffield)
* [ctools](https://drupal.org/project/ctools)
* [diskfree](https://drupal.org/project/diskfree)
* [duo](https://drupal.org/project/duo)
* [email\_confirm](https://drupal.org/project/email%5Fconfirm)
* [diff](https://drupal.org/project/diff)
* [ejectorseat](https://drupal.org/project/ejectorseat)
* [field\_permissions](https://drupal.org/project/field%5Fpermissions)
* [flood\_control](https://drupal.org/project/flood%5Fcontrol)
* [guardr\_core](https://drupal.org/project/guardr%5Fcore)
* [hacked](https://drupal.org/project/hacked)
* [hide\_php\_fatal\_error](https://drupal.org/project/hide%5Fphp%5Ffatal%5Ferror)
* [inactive\_user](https://drupal.org/project/inactive%5Fuser)
* [libraries](https://drupal.org/project/libraries)
* [login\_history](https://drupal.org/project/login%5Fhistory)
* [login\_security](https://drupal.org/project/login%5Fsecurity) \- Core login errors are blocked, accounts are silently blocked after 3 unsuccessful logins, an ongoing security alert is triggers an alert after 35 failed logins within 30 minutes, and the last login timestamp is displayed on login.
* [logging\_alerts](https://drupal.org/project/logging%5Falerts)
* [mass\_pwreset](https://drupal.org/project/mass%5Fpwreset)
* [memcache](https://drupal.org/project/memcache)
* [paranoia](https://drupal.org/project/paranoia)
* [password\_policy](https://drupal.org/project/password%5Fpolicy) \- Sets a password complexity of at least 3-mixed characters, 24 password history limitation, a digit placement restriction, a minimum length of 8 characters, and disabling the use of the username as a password.
* [permission\_watchdog](https://drupal.org/project/permission%5Fwatchdog)
* [r4032login](https://drupal.org/project/r4032login)
* [realname](https://drupal.org/project/realname)
* [remove\_generator](https://drupal.org/project/remove%5Fgenerator)
* [revision\_all](https://drupal.org/project/revision%5Fall)
* [role\_watchdog](https://drupal.org/project/role%5Fwatchdog)
* [seckit](https://drupal.org/project/seckit)
* [security\_review](https://drupal.org/project/security%5Freview)
* [semiclean](https://drupal.org/project/semiclean)
* [session\_expire](https://drupal.org/project/session%5Fexpire)
* [session\_limit](https://drupal.org/project/session%5Flimit)
* [settings\_audit\_log](https://drupal.org/project/settings%5Faudit%5Flog)
* [tfa](https://drupal.org/project/tfa)
* [tfa\_basic](https://drupal.org/project/tfa%5Fbasic)
* [user\_registrationpassword](https://drupal.org/project/user%5Fregistrationpassword)
* [username\_enumeration\_prevention](https://drupal.org/project/username%5Fenumeration%5Fprevention)
* [view\_profiles\_perms](https://drupal.org/project/view%5Fprofiles%5Fperms)
* [x\_originating\_ip](https://drupal.org/project/x%5Foriginating%5Fip)
